# Modern Chat UI

A responsive and interactive chat application built using HTML, CSS, and JavaScript.

## Features

- Responsive design
- Dark and light mode
- Typing indicator
- Auto scrolling
- Message timestamps
- Local storage persistence
- Smooth animations
- Mobile-friendly interface
- Interactive chat experience

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)

## Run Locally

1. Download the project.
2. Open the project folder.
3. Launch index.html in your browser.

## Future Improvements
