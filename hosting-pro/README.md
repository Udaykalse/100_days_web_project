# CloudNova - Modern Cloud Hosting Landing Page

CloudNova is a modern, responsive cloud hosting landing page built using HTML, CSS, and JavaScript. It showcases a professional SaaS-style design with interactive features, smooth animations, responsive layouts, and enhanced user experience for cloud infrastructure businesses, hosting providers, and technology startups.

## Features

* Fully Responsive Design
* Modern SaaS UI/UX
* Dark Mode Toggle
* Sticky Navigation Bar
* Mobile Hamburger Menu
* Smooth Scrolling Navigation
* Animated Statistics Counter
* Interactive Hosting Cost Calculator
* FAQ Accordion
* Form Validation & Success Feedback
* Scroll-to-Top Button
* Glassmorphism Inspired Cards
* Clean Code Structure
* Cross-Browser Compatible

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/cloud-hosting-pro.git
```

2. Navigate to the project folder

```bash
cd cloud-hosting-pro
```

3. Open the project

Simply open `index.html` in your browser.

## Usage

* Explore cloud hosting services
* Calculate hosting costs using the pricing calculator
* Submit demo requests through the contact form
* Browse FAQs and platform features
* Switch between light and dark themes
