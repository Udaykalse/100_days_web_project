# NovaTooltip — Modern Interactive Tooltip Library

**Project Folder:** `nova-tooltip`  
**Description:** A professional, accessible tooltip system with smooth animations, dynamic positioning, and interactive controls. Perfect for modern web apps.

## ✨ Suggested Features Added

- **Smart positioning** with edge detection & automatic flip
- **Glass morphism design** (backdrop-filter blur + gradients)
- **Keyboard accessible** (focus + ARIA labels)
- **Delay control** (0–400ms adjustable)
- **Live interactive playground** to customize text, position & timing
- **Fully responsive** (mobile, tablet, desktop)
- **Smooth scale + fade animations** using CSS transforms
- **Arrow indicators** that follow tooltip direction
- **Resize & scroll observers** for real-time repositioning
- **WCAG friendly** (focus indicators, high contrast)

## 🚀 Quick Start

1. Download all files: `index.html`, `style.css`, `script.js`
2. Keep them in the same folder (`nova-tooltip/`)
3. Open `index.html` in any modern browser

## 🎮 How to Use

- Hover or focus (Tab key) any button to see the tooltip
- Use the interactive panel to change tooltip text, position, and show delay
- Tooltips automatically avoid viewport edges
- Works on touch devices (tap + hold shows tooltip)

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, backdrop-filter, keyframes)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)

## 📱 Responsive Breakpoints

- Desktop: 1280px+ (grid layout)
- Tablet: 720px–1280px
- Mobile: <720px (stacked layout)

## ♿ Accessibility

- `role="tooltip"` and `aria-hidden` toggling
- `:focus-visible` outline on interactive elements
- Reduced motion preference respected
- Semantic HTML + ARIA descriptions
